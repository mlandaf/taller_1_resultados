## Descripción

Aplicación web para procesar y visualizar los resultados de la tarea de categorización de expresiones emocionales implementada en PsychoPy Builder durante el Taller 1.

No requiere instalar ningún software. Funciona directamente en el navegador.

---

## Cómo usar

1. Completar la tarea en PsychoPy Builder
2. Abrir la aplicación en el navegador usando el enlace de arriba
3. Arrastrar el archivo `.csv` generado por PsychoPy a la zona de carga
4. Seleccionar las columnas correspondientes en la pantalla de confirmación
5. Ver los resultados

---

## Qué muestra la aplicación

| Sección | Contenido |
|---|---|
| Resumen global | Precisión (%), RT promedio (ms) y total de trials |
| Por emoción | Precisión y RT para alegría, tristeza, ira y miedo |
| Comparación visual | Gráficos de barras de precisión y RT por emoción |
| Tabla de trials | Registro completo de todos los ensayos experimentales |

> **Nota:** Solo se procesan los trials de la fase experimental. La fase de práctica no se incluye en el análisis.
